# 프론트엔드 팀 스터디 가이드 - 4주 완성

> **목표**: HTML/CSS 경험자가 React + Next.js 프로젝트 참여 가능한 수준 달성  
> **기간**: 4주 (주 2회, 월/목)  
> **1회 학습시간**: 2-3시간  
> **총 학습시간**: 16-24시간  

## 📋 전체 학습 로드맵

| 주차 | 스케줄 | 주제 | 핵심 목표 |
|------|--------|------|-----------|
| **1주차** | 월/목 | JavaScript 핵심 기초 | 변수, 함수, 배열, 객체 마스터 |
| **2주차** | 월/목 | React 컴포넌트와 상태 | 컴포넌트, Props, State, 이벤트 처리 |
| **3주차** | 월/목 | API 연동과 라우팅 | 데이터 페칭, useEffect, React Router |
| **4주차** | 월/목 | Next.js와 실전 프로젝트 | Next.js 기초, 실제 프로젝트 구조 |

## 📚 1주차: JavaScript 핵심 기초

### Day 1 (월요일) - JavaScript 문법 기초
**🎯 학습 목표**: 변수, 분기, 반복문 완전 정복
- 변수 선언 (const, let)
- 조건문 (if-else, 삼항연산자, switch)
- 반복문 (for, while, for...of)
- 기초 실습: 티켓 재고 관리 프로그램

### Day 2 (목요일) - 배열, 함수, 객체
**🎯 학습 목표**: 고급 JavaScript 문법 마스터
- 배열 메서드 (map, filter, find, reduce)
- 함수 (일반함수, 화살표함수, 고차함수)
- 객체 조작 및 구조분해할당
- 종합 실습: 티켓 예약 시스템

---

## ⚛️ 2주차: React 컴포넌트와 상태

### Day 3 (월요일) - React 기초와 JSX
**🎯 학습 목표**: React 개발환경과 컴포넌트 기초
- React 프로젝트 생성 및 구조 이해
- JSX 문법과 기본 렌더링
- 컴포넌트 생성 및 Props 전달
- 실습: 티켓 카드 컴포넌트

### Day 4 (목요일) - 상태 관리와 이벤트
**🎯 학습 목표**: useState와 이벤트 처리 완전 정복
- useState 훅으로 상태 관리
- 이벤트 처리 (onClick, onChange)
- 폼 입력 다루기
- 조건부 렌더링과 리스트 렌더링
- 실습: 장바구니 기능이 있는 티켓 예약 앱

---

## 🌐 3주차: API 연동과 라우팅

### Day 5 (월요일) - useEffect와 API 연동
**🎯 학습 목표**: 실제 데이터로 동적 앱 만들기
- useEffect 훅 기초와 생명주기
- Fetch API로 데이터 가져오기
- 로딩 상태 및 에러 처리
- 실습: 외부 API 연동 티켓 목록

### Day 6 (목요일) - React Router와 폼 처리
**🎯 학습 목표**: 페이지 네비게이션과 데이터 전송
- React Router 설치 및 기본 라우팅
- 동적 라우팅 (URL 파라미터)
- 폼 제출과 유효성 검사
- Context API 기초 (전역 상태 관리)
- 실습: 완전한 티켓 예약 시스템

---

## 🚀 4주차: Next.js와 실전 프로젝트

### Day 7 (월요일) - Next.js 기초
**🎯 학습 목표**: Next.js 개발환경과 App Router
- Next.js 프로젝트 생성 및 구조
- App Router 파일 기반 라우팅
- 서버 컴포넌트 vs 클라이언트 컴포넌트
- 실습: 다중 이벤트 페이지 구조

### Day 8 (목요일) - 실전 프로젝트 완성
**🎯 학습 목표**: 프로젝트 참여 가능한 수준 달성
- API Routes로 백엔드 연동
- 실제 프로젝트 폴더 구조
- CSS 스타일링과 반응형 디자인
- 배포 및 최적화 기초
- 최종 프로젝트: 티켓 예약 시스템 완성
