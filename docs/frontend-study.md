## 📚 1주차: JavaScript 기초 문법

**🎯 학습 목표**: 프로그램의 흐름 제어하기

- for, while 반복문
- for...in, for...of 차이점
- break, continue 사용법
- 실습: 티켓 재고 관리 프로그램
