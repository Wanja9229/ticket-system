# 파이썬 6회차 과제: 간단한 은행 시스템 만들기

## 📋 과제 개요
클래스를 활용하여 고객과 계좌를 관리하는 간단한 은행 시스템을 만듭니다.

## 🎯 구현해야 할 클래스

### 1. Account 클래스 (계좌)

#### 속성
- `acc_no`: 계좌번호 (문자열)
- `balance`: 잔액 (정수, 기본값 0)

#### 메서드
- `__init__(self, acc_no, balance=0)`: 계좌번호와 초기 잔액을 받아 초기화
- `deposit(self, amount)`: 입금 기능
  - 금액이 0보다 크면 잔액에 추가하고 True 반환
  - 그렇지 않으면 False 반환
- `withdraw(self, amount)`: 출금 기능
  - 금액이 0보다 크고 잔액 이하면 잔액에서 차감하고 True 반환
  - 그렇지 않으면 False 반환
- `__str__(self)`: 계좌 정보를 "계좌: {계좌번호}, 잔액: {잔액}원" 형식으로 반환

### 2. Customer 클래스 (고객)

#### 속성
- `name`: 이름 (문자열)
- `birth`: 생년월일 (문자열, YYYYMMDD 형식)
- `accounts`: 보유 계좌 리스트 (빈 리스트로 시작)

#### 메서드
- `__init__(self, name, birth)`: 이름과 생년월일을 받아 초기화
- `add_account(self, balance=0)`: 새 계좌 개설
  - 계좌번호는 "{생년월일}-{순번}" 형식으로 자동 생성 (예: 19900101-01)
  - Account 객체를 생성하여 accounts 리스트에 추가
  - 생성된 계좌 객체 반환
- `get_account(self, acc_no)`: 계좌번호로 계좌 찾기
  - 해당 계좌가 있으면 계좌 객체 반환
  - 없으면 None 반환
- `total_balance(self)`: 모든 계좌의 잔액 합계 반환
- `__str__(self)`: 고객 정보를 "{이름} ({생년월일}), 계좌: {계좌수}개" 형식으로 반환

## 💻 간단한 메뉴 시스템 구현

### 필요한 변수
- `customers`: 고객들을 저장할 딕셔너리 (키: 이름, 값: Customer 객체)

### 메뉴 구성
```
1. 고객 등록
2. 계좌 개설
3. 입금
4. 출금
5. 잔액 조회
0. 종료
```

### 각 메뉴별 동작

#### 1. 고객 등록
- 이름과 생년월일 입력받기
- Customer 객체 생성하여 customers 딕셔너리에 저장
- 등록 완료 메시지 출력

#### 2. 계좌 개설
- 고객 이름 입력받기
- 해당 고객이 있으면 초기 입금액 입력받기
- add_account 메서드로 계좌 생성
- 생성된 계좌 정보 출력

#### 3. 입금
- 고객 이름과 계좌번호 입력받기
- 해당 계좌를 찾아서 입금액 입력받기
- deposit 메서드로 입금 처리
- 처리 결과와 현재 잔액 출력

#### 4. 출금
- 고객 이름과 계좌번호 입력받기
- 해당 계좌를 찾아서 출금액 입력받기
- withdraw 메서드로 출금 처리
- 성공 시 현재 잔액 출력, 실패 시 잔액 부족 메시지 출력

#### 5. 잔액 조회
- 고객 이름 입력받기
- 고객 정보와 모든 계좌 정보 출력
- 전체 잔액 합계 출력
