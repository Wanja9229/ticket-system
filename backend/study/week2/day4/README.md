# 과제4 – 회원 관리 시스템 프로그램 개발

## 📋 과제 개요

Python의 모듈, 파일 입출력, 예외 처리 기능을 종합적으로 활용하여 간단한 회원 관리 시스템을 개발해보세요. 이 프로그램은 회원 정보를 관리하고, 파일로 저장하며, 기본적인 데이터 관리 기능을 제공합니다.

## 🎯 프로그램 요구사항

프로그램은 `while` 반복문을 사용한 메뉴 기반 시스템으로 구성되어야 하며, 다음 6가지 주요 기능을 제공해야 합니다:

### 1. 회원가입 기능 (Create)

사용자로부터 회원 정보(이름, 생년월일, 등록일, 패스워드)를 입력받아 딕셔너리 형태로 임시 저장하는 기능을 구현하세요. 등록일은 현재 날짜를 자동으로 설정하도록 `datetime` 모듈을 활용하고, 입력받은 모든 정보는 메모리상의 딕셔너리에 보관합니다.

### 2. 파일 저장 기능

메모리에 임시 저장된 **모든 회원정보를 하나의 JSON 파일**에 저장하는 기능을 만드세요. 기존 파일이 있다면 덮어쓰기 방식으로 처리하며, `with` 문을 사용하여 안전한 파일 처리를 구현해야 합니다. **여러 명의 회원 데이터를 하나의 파일에 리스트 형태로 저장**합니다.

### 3. 회원 정보 조회 기능 (Read)

저장된 JSON 파일에서 특정 회원의 정보를 불러와 화면에 출력하는 기능을 개발하세요. 사용자가 파일명과 회원 이름을 입력하면 해당 회원의 상세 정보를 표시합니다.

### 4. 전체 회원 목록 조회 기능 (List)

저장된 JSON 파일에서 **모든 회원의 정보를 불러와 목록 형태로 출력**하는 기능을 구현하세요. 회원 번호, 이름, 생년월일, 등록일을 표 형태로 보기 좋게 정리하여 출력합니다. 비밀번호는 보안상 출력하지 않거나 마스킹 처리합니다.

### 5. 회원 정보 수정 기능 (Update)

기존 회원의 정보를 수정하는 기능을 개발하세요. 사용자가 회원 이름을 입력하면 해당 회원을 찾아 생년월일이나 패스워드 등의 정보를 수정할 수 있도록 합니다. 수정 후에는 변경된 내용을 파일에 다시 저장해야 합니다.

### 6. 파일 삭제 기능 (Delete)

사용자가 입력한 파일명에 해당하는 회원 데이터 파일을 시스템에서 완전히 삭제하는 기능을 구현하세요. 파일 삭제 전 확인 메시지를 출력하고, 삭제 결과를 사용자에게 알려줍니다.

## 🔧 기술적 구현 조건

- **모듈 활용**: `random`, `datetime`, `os`, `json` 등 표준 라이브러리를 적절히 import하여 사용
- **함수 분리**: 각 기능별로 helper 함수를 정의하여 코드의 재사용성과 가독성을 높임
- **예외 처리**: 파일 입출력 시 발생할 수 있는 `FileNotFoundError`, `PermissionError` 등을 `try-except` 구문으로 처리
- **JSON 형식**: 회원 데이터는 JSON 형식으로 저장하여 구조화된 데이터 관리
- **다중 데이터 저장**: 하나의 파일에 여러 명의 회원 정보를 리스트 형태로 저장
- **CRUD 완성**: Create, Read, Update, Delete 기능을 모두 구현하여 완전한 데이터 관리 시스템 구축
- **사용자 인터페이스**: 명확한 메뉴 옵션과 안내 메시지를 제공하여 사용자 경험을 개선
- **데이터 검증**: 입력받은 데이터의 유효성을 검사하고 적절한 피드백 제공
