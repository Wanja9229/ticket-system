## 🚀 시작하기

1. Python 3.11+ 설치
2. 가상환경 생성 및 활성화
3. 필요 패키지 설치

```bash

# 폴더 이동
cd ~/ticket-system/backend/study

# pip 업그레이드 
python.exe -m pip install --upgrade pip

# 가상환경 생성
python -m venv venv

# 활성화 (Windows)
venv\Scripts\activate

# 활성화 (Linux/Mac/WSL)
source venv/bin/activate

# 패키지 설치
pip install -r requirements.txt
```
