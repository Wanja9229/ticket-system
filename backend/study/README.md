## ✅ 2회차 – 리스트/딕셔너리 고급 & 함수형 내장함수

### 📚 학습 범위
- 리스트 메서드 : append, remove, sort, etc.
- 딕셔너리 메서드 : keys(), values(), get()
- `map()`, `filter()`, `zip()`, `enumerate()`
- 리스트 컴프리헨션

### 🧪 다음 회차까지 할 과제
```python 
nums = [4,5,7,8]
```
- 주어진 배열에서 4 앞에 1,2,3 의 값 추가하기(ex [1,2,3,4,5,7,8])
- 주어진 배열에서 8뒤에 9 값 추가하기 (ex [1,2,3,4,5,7,8,9])
- 주어진 배열에서 짝수만 추출 후 출력 - 컴프리헨션으로 안하기
- 주어진 배열에서 filter() 활용해서 짝수만 추출 후 출력 - 컴프리헨션으로 하기
- 주어진 배열에서 컴프리헨션으로 숫자 역정렬 출력
- 주어진 배열에서 map() 내장함수 활용해서 모든 숫자 +10 해서 출력하기
```python 
students = {
    "철수": 87,
    "영희": 92,
    "민수": 49,
    "지민": 73,
    "수지": 65,
    "정우": 91,
    "하늘": 58,
    "민지": 100,
    "도윤": 77,
    "예린": 84
}
# enumerate 로 for문 작성 
# 등급별 표시 (이름 : 점수(등급) ) EX: 철수:87점(B)
# 그냥 출력
# 점수순 출력
# 이름 가나다순 출력
```

### 🧠 핵심 키워드
- `map()` : 각 요소에 함수 적용
- `filter()` : 조건 만족 요소만 필터링
- `enumerate()` : 인덱스 포함 반복
- 리스트 컴프리헨션 : 간결한 리스트 생성 문법
